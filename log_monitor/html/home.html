Home

Welcome to the documentation for Brand Nudge Data Collect tools and helpers, a comprehensive collection of utilities designed to simplify and enhance the process of web scraping with Python. Whether you’re a beginner or an experienced scraper, our tools aim to streamline your workflow and help you efficiently extract data from websites.

Note

This project is under active development.

Key Features

  1. Robust HTML Parsing: Easily navigate and parse complex HTML structures with our intuitive helpers.

  2. HTTP Requests: Simplified methods for making HTTP requests, handling sessions, and managing headers and cookies.

  3. Data Extraction: Tools for extracting data from web pages using CSS selectors, XPath, and regular expressions.

  4. Rate Limiting: Built-in support for rate limiting to avoid overloading servers and to comply with website policies.

  5. Proxy Management: Seamlessly integrate proxy support to bypass IP blocking and access geo-restricted content.

  6. Data Storage: Utilities for storing scraped data in various formats such as CSV, JSON, and databases.

  7. Error Handling: Comprehensive error handling to manage and debug issues during scraping.

  8. Extensibility: Easily extend and customize tools to fit your specific scraping needs.

Example Project Usage

This project has a standard AWS EC2 layout which is built by the Data Collect team and can be build locally on your own laptop.

You can build and view this documentation project locally - we recommend that you activate a local Python virtual environment first:

# Install required Python dependencies
pip install -r requirements.txt

# Enter the core directory
cd core/

# Run a collection
python supervalu.main.py

Update The Docs

Run below command to update the current documentation

# Enter the docs directory
cd core/

# Generate the new html documentation
make html
