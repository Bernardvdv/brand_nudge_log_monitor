john_lewis_reviews_keywords.extract_keywords(text)[source]

In this code, we first tokenize the input text into words using word_tokenize function from nltk. We then remove stop words and punctuation marks using stopwords module and a list comprehension.

Next, we calculate the frequency distribution of the remaining words using FreqDist function from nltk. Finally, we extract the main keywords from the frequency distribution by selecting words with a length longer than 4 characters and return them as a set.

Note that the output keywords are in lower case and in a set data structure, which removes duplicates. You can adjust the code to your needs, such as changing the minimum keyword length or returning a list instead of a set. :param text: :return:
